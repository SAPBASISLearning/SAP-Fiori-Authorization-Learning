// 01 — FOUNDATION

S/4HANA Authorization Layers

Unlike ECC which has one layer, S/4HANA has THREE independent authorization layers. A user needs to pass ALL three to access an app.

LAYER 01
ABAP Buffer
Standard auth objects checked at runtime
LAYER 02
HANA DB
Database privilege check at DB level
LAYER 03
Fiori Launchpad
/UI2/ objects — tile & catalog access
